## 安装使用（Installation）

``` sh
# 进入文件目录
cd source

# 启动
npm run dev

# 新建文件夹
hexo new page <文件夹名称>

# 新建文件
hexo new post <文章标题>

# 部署
npm run build

# 提交
npm run upload <提交信息说明>
```
